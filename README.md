# 🧱 Brick by Brick – LEGO Analytics Dashboard

> An end-to-end Business Intelligence project built with **Power BI**, **Power Query**, and **DAX** to transform raw LEGO data into an interactive analytics dashboard.

---

# 📖 Project Overview

This project demonstrates the complete Business Intelligence workflow, from data preparation and transformation to interactive dashboard development.

Using Power BI, I cleaned and transformed the LEGO dataset, created calculated columns and DAX measures, and designed an interactive dashboard that enables users to explore LEGO sets through dynamic filtering, drill-down analysis, and advanced report interactions.

---

# 🎯 Project Objectives

## 📌 Data Preparation

- Imported the LEGO dataset using Power Query
- Removed unnecessary columns
- Corrected data types
- Filtered incomplete records
- Created custom **Age Range** categories
- Created custom **Price Range** categories

---

## 📌 Data Modeling

Created DAX measures to calculate:

- Total Sets
- Total Theme Groups
- Average Price
- Average Pieces
- Average Age

---

## 📌 Dashboard Development

Designed an interactive report featuring:

- KPI Cards
- Dynamic Table
- Slicers
- Set Details Panel
- Cross-filtering
- Responsive Layout

---

## 📌 Advanced Power BI Features

Implemented:

- What-If Parameter
- Dynamic Price Filtering
- Image Tooltips
- Bookmarks
- Reset Filters Button
- Page Navigation
- Decomposition Tree
- Custom Visual Interactions
